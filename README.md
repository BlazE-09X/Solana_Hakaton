# 🏠 ProofRent

ProofRent — это Web3-приложение на базе Solana, которое позволяет реализовать долевое владение реальными активами (RWA), такими как недвижимость.

## Проблема

Объекты недвижимости и другие реальные активы обладают рядом недостатков:
- **Высокий порог входа:** инвестиции требуют огромных капиталов.
- **Сложность передачи:** бюрократия и долгий процесс переоформления.
- **Отсутствие прозрачности:** сложно отследить историю владения.
- **Низкая ликвидность:** актив трудно быстро продать.

## 💡 Решение

ProofRent токенизирует недвижимость, превращая её в токены на блокчейне, что обеспечивает:
- **Долевое владение:** возможность купить лишь часть объекта.
- **Прозрачность:** все транзакции записываются в блокчейн.
- **Глобальный доступ:** инвестиции доступны из любой точки мира.
- **Простота передачи:** мгновенная смена владельца через блокчейн.

## ⚙️ Как это работает

1. Владелец токенизирует актив, создавая **SPL-токены**.
2. Токены представляют собой доли (акции) объекта недвижимости.
3. Пользователи подключают кошелек (например, **Phantom**).
4. Пользователи покупают токены.
5. Право владения обновляется непосредственно в блокчейне (on-chain).

## 🔗 Технологический стек

- **Blockchain:** Solana (Devnet)
- **Standard:** SPL Tokens
- **Frontend:** React + TypeScript
- **Backend:** Node.js
- **Wallet:** Phantom Wallet

## Функционал (MVP)

- Подключение кошелька (Phantom).
- Отображение баланса **SOL**.
- Отображение баланса **SPL-токенов** проекта.
- Покупка токенов (механика трансфера).
- Выполнение транзакций в сети Solana.

## Архитектура

**Frontend:**
- React (интерфейс)
- Solana Web3.js (взаимодействие с блокчейном)

**Backend:**
- Node.js
- Логика перевода SPL-токенов

**Blockchain:**
- Solana Devnet
- Token Mint (выпуск токенов стандарта SPL)
