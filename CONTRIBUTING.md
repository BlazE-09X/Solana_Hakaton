# Contributing to ProofRent

Спасибо за интерес к проекту!

## Как внести вклад
1. Форкните репозиторий.
2. Создайте ветку: `git checkout -b feat/new-feature`.
3. Сделайте коммит: `git commit -m 'Add some feature'`.
4. Пушните в ветку: `git push origin feat/new-feature`.
5. Откройте Pull Request.

## Требования
- Наличие установленного **Anchor CLI** и **Solana CLI**.
- Весь код на Rust должен проходить `anchor build`.
